# RHEL Systemd Daemon Workload Lab

## Overview
THis lab demonstrates how to create, manage, and secure system services using systemd.

## What This Lab Covers
- Custom systemd service creation
- Environment file usage
- Restart policies and dependencies
- systemd timer (cron replacement)
- Service minimization (DISA STIG alignment)
- systemd hardening controls
- Scheduled task auditing and restriction

## Environment
- RHEL 9.5
- systemd

## Key Concepts
- Separation of configuration and execution
- Service resilience via restart policies
- Secure service isolation using systemd directives
- Replacement of cron with systemd timers
